The Claude Skills Playbook I Wish I Had on Day One

From: AI Maker (One Shot Show) Apr 26, 2026 Label: AI AI Summary 
Episode 7 of One Shot Show features Ilia Karelin (Prosper newsletter) walking through Claude Skills live. The core rule for identifying skill candidates: 'What am I doing too often manually?' — if you repeat something 3+ times a week, convert it to a skill. Skills are reusable instruction sets stored as SKILL.md files that Claude pulls in automatically; the Skill Creator meta-skill (pre-installed in Claude Desktop/Cowork) can scaffold a skill in ~5 minutes. The live demo built a 'second-opinion' skill that pushes back on ideas from multiple perspectives. Post includes Ilia's Fast Research skill (packaged for install) which produces a sourced research brief saved as Markdown. Key insight: time spent on skills is mostly in evaluating output quality, not building them. Four-element prompt structure for the Skill Creator is covered in the full post.
