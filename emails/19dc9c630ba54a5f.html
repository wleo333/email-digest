The Claude Skills Playbook I Wish I Had on Day One

From: AI Maker (One Shot Show) Apr 26, 2026 Label: AI AI Summary 
Episode 7 of One Shot Show featured AI expert Ilia Karelin walking through Claude Skills 2.0. The core rule for building skills: do it when you catch yourself doing the same thing manually 3+ times a week. Ilia demonstrated the Skill Creator (a meta-skill pre-installed in Claude Desktop/Cowork) by building a 'second opinion' skill live — a skill that pushes back on ideas from multiple perspectives rather than defaulting to agreement. The four-element prompt structure for the Skill Creator: what you want Claude to do, your use case, audience details, and examples. The post includes: the full 60-min session recording, Ilia's Fast Research skill packaged for install (gives a sourced research brief as Markdown, personalized via a first-run interview), and a full build walkthrough. Key note: Claude Code users must install plugins first before using the Skill Creator; it's pre-installed on Claude Desktop and Cowork.
